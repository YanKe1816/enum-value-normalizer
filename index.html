Enum Value Normalizer

This app normalizes a raw free-text value into one of a fixed list of allowed enum values.

This tool provides deterministic enum value normalization for workflows that need to pass fixed values into downstream systems.

It does not create new enum values, make business decisions, call external systems, or store user data.

What It Does

  • Normalizes free-text values into fixed allowed enum values.
  • Handles simple formatting differences such as case, spaces, underscores, and hyphens.
  • Returns a fixed structured result with normalized_value, is_supported, confidence, status, and error.

When This Tool Helps

  • When a downstream system only accepts fixed enum values.
  • When user text needs to be mapped to one of the allowed options.
  • When a workflow needs deterministic value normalization instead of open-ended interpretation.

What It Does Not Do

  • It does not create new enum values.
  • It does not make business decisions.
  • It does not explain business meaning.
  • It does not call external APIs or store request data.

MCP Endpoint

The MCP endpoint is POST /mcp. Health is available at GET /health.
